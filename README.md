# 🕸️ SpiderSense 蜘蛛感应

> 像蜘蛛一样思考的AI Agent

## 简介

SpiderSense（蜘蛛感应）是一个多节点分布式思考系统，让AI能够自动感知丝线震动、联动节点、调动能力。

## 核心理念

- **连接**：多节点多能力网络
- **感应**：场景触发，全局联动
- **思考**：多节点分布式思考
- **中枢**：快速判断，调动技能网

## 功能特性

- ✅ 11个丝线震动检测
- ✅ 10个能力节点联动
- ✅ 中枢快速决策
- ✅ 多节点思考（调研部/记忆部/执行部/创意部/审批部）
- ✅ 角色代入（打印助手/民宿管家/小红书运营/商户拓展）

## 快速开始

```bash
git clone https://github.com/yilin990/spider-sense.git
cd spider-sense
python3 run.py <任务>
```

## 示例

```bash
python3 run.py 截图
python3 run.py 帮我搜索AI最新资讯
python3 run.py 我要发帖
```

## 丝线（触发词）

| 丝线 | 触发场景 |
|------|---------|
| 截图 | 需要截图时 |
| 搜索 | 需要查资料时 |
| 发帖 | 需要发布内容时 |
| 记忆 | 需要记住信息时 |
| 商户 | 需要商户管理时 |
| 打印 | 需要打印时 |
| 看图 | 需要分析图片时 |
| 语音 | 需要语音合成时 |
| 飞书 | 需要发通知时 |
| 代码 | 需要写脚本时 |
| 文件 | 需要文件操作时 |

## 架构

```
丝线层（场景触发）
    ↓
节点层（能力网络）
    ↓
中枢层（判断+执行）
```
